### 3. 本地缓存优化

#### 优化前
- 每次都调用API获取数据
- 缓存机制不够完善

#### 优化后
- ✅ 优先读取本地缓存文件
- ✅ 缓存有效期7天
- ✅ 支持强制刷新参数
- ✅ 自动保存API获取的数据

#### 缓存机制
```python
# 检查缓存文件是否存在且未过期
cache_file = os.path.join(CACHE_DIR, f"{stock_code}.csv")
if os.path.exists(cache_file) and not force_refresh:
    # 检查文件是否过期（7天）
    file_time = os.path.getmtime(cache_file)
    if time.time() - file_time < 7 * 24 * 3600:
        # 使用缓存数据
        return pd.read_csv(cache_file)
```

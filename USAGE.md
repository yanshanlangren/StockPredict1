# 股票交易系统使用指南

## 环境要求

- **Python版本**: >= 3.8
- **内存**: 4GB+ 推荐
- **磁盘**: 2GB+ 可用空间

## 安装步骤

### 1. 安装依赖

```bash
# 升级pip
pip install --upgrade pip

# 安装依赖
pip install -r requirements.txt
```

### 2. 启动服务

```bash
python app.py
```

看到以下提示表示启动成功：
```
股票交易AI系统（全局模型版）启动中...
* Running on http://0.0.0.0:5000/
```

### 3. 访问应用

浏览器打开 http://localhost:5000/

---

## 功能使用

### 1. 训练模型

**页面**: 首页

**步骤**:
1. 查看当前模型状态
2. 配置训练参数：
   - 股票数量（建议50-200）
   - 历史天数（建议200-300）
   - 训练轮数（建议50-100）
3. 点击"开始训练"
4. 等待训练完成

**命令行方式**:
```bash
python train_global_model.py --stocks 100 --days 300 --epochs 100
```

### 2. 批量预测

**页面**: 批量预测

**步骤**:
1. 配置预测参数：
   - 返回数量（Top N）
   - 最低价格
   - 最高价格
2. 点击"开始预测"
3. 查看结果：
   - 预测上涨的股票优先展示
   - 按预期收益率排序
   - 显示置信度和预测价格

### 3. 单股预测

**页面**: 单股预测

**步骤**:
1. 输入或选择股票代码
2. 点击"预测"
3. 查看结果：
   - 预测方向（上涨/下跌）
   - 预期收益率
   - 置信度

### 4. 股票查询

**页面**: 股票

**步骤**:
1. 选择股票代码
2. 点击"查询"
3. 查看K线图和统计信息

## 常见问题

### Q1: 模型未训练怎么办？

**解决方案**: 在首页点击"开始训练"按钮，或运行：
```bash
python train_global_model.py
```

### Q2: 数据获取失败？

**解决方案**: 
- 检查网络连接
- 本地缓存有效期为7天，过期自动刷新
- 可以强制刷新股票列表

### Q3: 预测结果为下跌？

**解决方案**: 
- 这是正常的，模型预测基于当前市场状态
- 预测结果仅供参考，不构成投资建议

### Q4: 训练时间太长？

**解决方案**:
- 减少训练股票数量
- 减少训练轮数
- 减少历史天数

## API使用示例

### 健康检查
```bash
curl http://localhost:5000/api/health
```

### 获取模型信息
```bash
curl http://localhost:5000/api/model/info
```

### 单股预测
```bash
curl -X POST http://localhost:5000/api/predict/600000 \
  -H "Content-Type: application/json" \
  -d '{"days": 100}'
```

### 批量预测
```bash
curl -X POST http://localhost:5000/api/predict/batch \
  -H "Content-Type: application/json" \
  -d '{"top_n": 20, "min_price": 0, "max_price": 100}'
```

## 性能优化建议

1. **训练优化**：
   - 使用GPU加速TensorFlow
   - 增加训练股票数量提高泛化能力

2. **预测优化**：
   - 批量预测比单股预测更高效
   - 本地缓存可减少网络请求

3. **系统优化**：
   - 定期清理过期缓存
   - 监控系统内存使用
